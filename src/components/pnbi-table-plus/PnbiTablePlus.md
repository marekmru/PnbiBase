### Usage

Usually used inside a pnbi-page with headline as VIP or title as secondary Table Element

```jsx
<pnbi-datatable headline="Tablename" button-label="New project" >
  <!-- Optional Slot-->
  <div slot="primary-controls">
  </div>
  <!-- Optional Slot-->
  <div slot="secondary-controls">
  </div>
  <!-- Default Slot-->
  <v-data-table >
        <!-- See Vuetify-->
  </v-data-table>
</pnbi-datatable>
```
